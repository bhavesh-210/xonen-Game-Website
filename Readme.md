# 🎮 Xonen Game Website

I built this as a passion project — a gaming-themed website inspired by the Zentry universe. It's got a full-screen hero video, multiple pages for different parts of the experience, and a vibe that actually feels like a gaming brand, not just another template.

No frameworks. No build tools. Just HTML, CSS, and vanilla JS doing the heavy lifting.

🌐 **Check it live:** [xonen-game-website.vercel.app](https://xonen-game-website.vercel.app)

---

## 💡 What's this project about?

Think of it as a frontend concept for a gaming platform called Zentry — where players from across web2 and web3 games come together in one shared universe. The site brings that vision to life visually: immersive videos, clean card layouts, a social hub (Nexus), a game library (Vault), and a contact section that doesn't feel like a boring form.

It was a great exercise in building something that *feels* premium using just the basics.

## 🗂️ How it's organized

```
xonen-Game-Website/
├── index.html        # The main landing page — hero, about, feature cards, footer
├── products.html     # Products listing
├── nexus.html        # Nexus — the social hub page
├── vault.html        # Vault — game archive/library
├── style.css         # All the styling lives here
├── script.js         # Interactions for the home page
├── nexus.js          # Nexus page logic
├── product.js        # Products page logic
├── vault.js          # Vault page logic
├── assets/
│   ├── img/          # Images, logos, webp files
│   └── videos/       # Hero video + feature card videos
└── gitignore
```

## ✨ What's inside

- **Hero Section** — Fullscreen autoplay video with a bold tagline. Sets the tone immediately.
- **About Section** — A quick intro to the Zentry world and what it's building.
- **Feature Cards** — Upcoming products: Radiant, Zigma, Nexus, and Azul — each with its own video loop.
- **Nexus Page** — A gamified social hub concept for Web3 gaming communities.
- **Vault Page** — Where your games live. A clean archive/library view.
- **Products Page** — Browse the featured lineup.
- **Contact Popup** — A smooth modal instead of a boring contact page. Clicks open, emails show up, done.
- **Responsive Layout** — Works on mobile too, with a hamburger menu that actually works.

## 🚀 Run it yourself

There's nothing to install. Seriously.

```bash
git clone https://github.com/bhavesh-210/xonen-Game-Website.git
cd xonen-Game-Website
```

Then just open `index.html` in your browser. Or if you want live reload while editing, use VS Code's Live Server extension or:

```bash
# If you have live-server installed
live-server
```

That's it.
